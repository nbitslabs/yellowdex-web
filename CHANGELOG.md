# Changelog

All notable changes to the Yellowdex landing page will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Automated release workflow with semantic versioning
- Changelog file to track notable changes
- Enhanced CI workflow with build artifact caching
- PR template for standardized pull request descriptions

### Changed
- Deploy workflow now includes deployment metadata and build-time token for GitHub API
- CI workflow now uploads build artifacts for potential reuse

## [0.1.0] - 2025-01-XX

### Added
- Initial landing page for Yellowdex browser extension
- Hero section with tagline and extension detection
- Features showcase with interactive overlays
- FAQ section with common questions
- Download CTA with latest release fetching from yellowdex-ext repo
- Responsive design with Tailwind CSS v4
- GitHub Pages deployment via GitHub Actions
